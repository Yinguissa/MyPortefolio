MY Projects

CNT Human Resource Management System(Guinea)

I developed a Java Swing desktop application for managing human resources at the Conseil National de la Transition (CNT). The system allows efficient management of employee records, contracts, leave requests, and department assignments. It features an intuitive user interface, database integration with MySQL, and role-based access control to ensure data security..

Delivery parcel and marchandises transportation app

I developed a Flutter-based delivery app for e-Cash, designed to facilitate parcel and merchandise transportation. The app enables users to book deliveries, track shipments in real time, and manage transactions securely. It features a user-friendly interface, GPS tracking, payment integration, and order history management, ensuring a seamless delivery experience. This project enhanced logistics efficiency and streamlined the shipping process for businesses and individuals.

Maaliando – Tontine Management App

I developed Maaliando, a tontine management application designed to simplify and digitalize traditional group savings. Built with Flutter, the app allows users to create and join savings groups, track contributions, manage payouts, and receive notifications. With a secure authentication system and a user-friendly interface, Maaliando ensures transparency and efficiency in community-based financial management. This project aimed to modernize informal savings practices and provide a reliable financial tool for users.

Postgraduate Student Management System – Gamal Abdel Nasser University(Guinea-SEAG)

I worked on developing a database system for the Advanced Studies Department at Gamal Abdel Nasser University. The project aimed to create an efficient system for managing master's and PhD students. It included features for student records management, enrollment tracking, academic progress monitoring, and administrative operations, ensuring a seamless and organized postgraduate student management process.

Stock and Delivery Management System – eCash

I developed a mobile application in Flutter for eCash, designed to facilitate the management of stock and deliveries. The system allowed businesses to track product entries and exits, manage inventory levels, and streamline the delivery process. It provided a user-friendly interface for real-time monitoring, ensuring efficient logistics and stock management.

Bank Management System – Spring Boot

I developed a Bank Management System using Spring Boot, designed to handle core banking operations efficiently. The system includes features for account creation, deposit and withdrawal transactions, and account deletion. It ensures secure and seamless financial transactions while maintaining a structured and scalable backend architecture. This project enhances digital banking solutions by providing a reliable and user-friendly system for financial institutions.
